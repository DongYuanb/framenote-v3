# FrameNote - AI视频笔记工具（生产版）

## 🚀 产品概述

FrameNote是一款基于AI技术的智能视频笔记工具，能够自动提取视频中的PPT内容、生成结构化讲义和知识图片，为教育、培训、学习等场景提供高效的视频内容处理解决方案。

## ✨ 核心功能

### 🎥 智能视频分析
- **AI驱动**: 基于先进的人工智能技术，自动识别和分析视频内容
- **多格式支持**: 支持MP4、AVI、MOV、MKV等主流视频格式
- **大文件处理**: 支持最大500MB的视频文件处理

### 📊 PPT一键提取
- **智能识别**: 自动识别视频中的PPT幻灯片内容
- **文字提取**: 精确提取PPT中的文字信息
- **图片保存**: 自动保存PPT中的图片素材

### 📝 讲义自动生成
- **结构化输出**: 根据视频内容自动生成层次化的讲义
- **内容整理**: 智能整理和分类视频中的知识点
- **格式优化**: 生成易于阅读和学习的文档格式

### 🖼️ 知识图片制作
- **概念可视化**: 将复杂概念转化为直观的知识图片
- **图表生成**: 自动生成思维导图和知识图谱
- **记忆辅助**: 提供视觉化的学习辅助工具

### 💳 会员体系
- **多层级服务**: 提供免费、基础、高级、专业四个等级
- **灵活计费**: 支持月卡、季卡、年卡多种计费方式
- **特权功能**: 不同等级享受不同的处理时长和功能特权

## 🚀 快速部署

### 环境要求
- **Python**: 3.8+ 
- **Node.js**: 16+
- **PostgreSQL**: 12+
- **Redis**: 6+
- **FFmpeg**: 视频处理工具（可选）

### 后端部署
```bash
# 1. 安装Python依赖
cd amazing_videodoc-main
pip install -r requirements.txt

# 2. 配置环境变量
cp env_example.txt .env
# 编辑 .env 文件，填入真实配置

# 3. 初始化数据库
python -c "from database import init_db; init_db()"

# 4. 启动服务
python main.py
```

### 前端部署
```bash
# 1. 安装依赖
cd zed-landing-vibe-main
npm install

# 2. 构建生产版本
npm run build

# 3. 预览构建结果
npm run preview
```

### Docker部署（推荐）
```bash
# 1. 构建镜像
docker build -t framenote .

# 2. 运行容器
docker run -d -p 8002:8002 --name framenote framenote

# 3. 查看日志
docker logs framenote
```
